org.java.controller  // -> service에서 일을 시키고 결과 반환 받는다-> 콘솔에 표시   1
org.java.service       // -> controller에서서 일 시킨걸 수행 하고 결과 값을 반환 해준다 2
org.java.dao	// ->service중에서 DB에 관련 일을 수행 하고 결과를 반환 해준다 3
org.java.dto               // ->  dao수행 중 DB에 데이터를 연결 시켜주는 역할을 수행 -> DB테이블과 구조같다 4
org.java.connect      // -> java와 데이터베이스(DB)를 연결 해주는 역할 5


service ->일 -> 회원관리 ->  CRUD
Creat -> 회원가입
Read  -> 회원조회
Update -> 회원수정
Delete  -> 회원삭제(탈퇴)

공통 -> 회원 관리 -> 매서드    excuteQueryService() 
공통 클래스 -> MemberService 클래스    excuteQueryService() 
클래스 생성 -> 상속 -> 공통 클래스

회원가입 -> MemberInsert -> MemberService 
	  excuteQueryService() 오버라이드 구현 "회원가입"
회원조회 -> MemberSelect-> MemberService 
	  excuteQueryService() 오버라이드 구현 "회원조회"
회원수정 -> MemberUpdate-> MemberService 
	  excuteQueryService() 오버라이드 구현 "회원수정"
회원탈퇴 -> MemberDelete -> MemberService 
	  excuteQueryService() 오버라이드 구현  "회원탈퇴"
