ctrl + shift + P -> 설정
ctrl + -> setting
ctrl + shift + ` -> 터미널 파워 셀 열기
ctrl + ` -> 터미널 열기, 닫기

Ctrl + 클릭 : 해당 함수 정의문, 링크로 이동
Ctrl + 1 or 2 : 분할화면 이동

Ctrl + W -> 현재파일 닫기
Ctrl + Shift + W -> 모든 파일 닫기
